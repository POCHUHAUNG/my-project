# Google 試算表操作說明

## 將 CSV 匯入 Google 試算表

1. 開啟 [sheets.google.com](https://sheets.google.com) → 新試算表
2. 上方選單：**檔案 → 匯入**
3. 選擇「**上傳**」→ 從桌面選取 `.csv` 檔案
4. 分隔符號選「**逗號**」或「**自動偵測**」
5. 按「**匯入資料**」
6. 出現「大功告成！」後點「**立即開啟 »**」

---

## 加上可打勾的核取方塊

1. 點 **A 欄標題**（選取整欄）
2. 上方選單：**插入 → 核取方塊**
3. 所有格子變成可點擊的 ☑ Box

---

## 將 Markdown 表格貼到 Google 試算表

1. 複製 Markdown 表格內容
2. 點選試算表 A1 格 → 貼上
3. 如果沒有自動分欄：
   - 選取貼上的資料
   - 上方選單：**資料 → 將文字分割成資料欄**
   - 分隔符號選「**Tab**」

---

## 從 Google 試算表複製到 Word（格式不亂）

1. 在試算表選取要複製的範圍
2. Ctrl+C 複製
3. 開啟 Word → 貼上時選「**保留來源格式**」
4. 表格會完整貼入，欄位對齊

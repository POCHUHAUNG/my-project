# 活動報名系統 — 功能模板

> 建立日期：2026-04-04  
> 部署：前端 Vercel + 後端 Render + 資料 Google Sheets

---

## 前台功能

### 活動首頁
- 主視覺圖片（hero banner）
- 課程 DM 圖片展示
- 活動標題、日期、地點、詳細說明
- 回到頂部浮動按鈕（滑動超過 300px 出現）

### 議程展示
- 單場次（單欄）/ 多場次（最多 3 欄）自動切換版型
- 議程項目：時間、主題、講師、說明
- 可自訂議程標題（中文 / 英文）
- 自動識別場次分類（上午 / 下午 / 晚上 / 全天 / Day1/Day2）

### 報名表單
- 必填欄位：姓名、Email、電話、公司 / 單位、LINE ID
- 即時驗證（Email 格式、必填欄位）
- 自動滾動至第一個錯誤欄位
- 報名成功後顯示臨時密碼
- 報名資料暫存（登入跳轉後可恢復）

### 會員系統
- **登入**：Email + 密碼，支援跳轉回原頁面
- **忘記密碼**：發送重設連結（24 小時有效），防止 Email 枚舉
- **設定密碼**：token 驗證，密碼強度規則（8 碼以上、含大小寫與數字）
- **個人頁面**：
  - 會員基本資訊
  - 報名紀錄（課程、日期、地點、出席狀態）
  - 報到 QR Code（掃碼即報到）
  - 更改密碼
  - 登出

### 右上角會員狀態欄
- 訪客：顯示「會員登入」按鈕
- 已登入：顯示姓名 + 頭像，下拉選單含報名資訊預覽、前往個人頁、登出

## 後台功能（`/admin`）

### 後台驗證
- 管理員密碼登入（`x-admin-password` header）
- 首次啟動提示（Render 喚醒最多 60 秒）

### 會員管理
- 查看清單：編號、姓名、Email、公司、帳號狀態、加入日期、課程名稱、上課日期、出席狀態
- 新增會員：姓名（必填）、Email（必填）、電話、公司、LINE ID、密碼、是否寄設定密碼信
- 編輯會員：姓名、Email、啟用狀態
- 刪除 / 清空所有會員（含確認步驟）

### 批次匯入
- CSV 匯入（欄位：name、email、phone、company、lineId）
- 逐筆驗證，回傳成功 / 失敗筆數

### 資料匯出
- 匯出 CSV / Excel
- 欄位：編號、姓名、Email、公司單位、帳號狀態、加入日期、課程名稱、上課日期、出席狀態

### 活動資訊編輯（需從後台進入主頁）
- 上傳 / 更新主視覺圖片
- 上傳 / 更新課程 DM
- 修改議程標題（中 / 英文）

### 活動通知
- 選擇類型：資訊更改 / 取消活動
- 自訂通知內容
- 發送至所有報名者，回傳已發送數量

---

## Email 通知（共 6 種）

| 觸發時機 | 收件人 | 內容 |
|---|---|---|
| 首次報名 | 會員 | 歡迎信 + 設定密碼連結（24h） |
| 忘記密碼 | 會員 | 重設密碼連結（24h） |
| 報名成功 | 報名者 | 活動資訊、會員編號、個人頁連結 |
| 新報名通知 | 主辦單位 | 報名者姓名、Email、電話、公司等 |
| 活動資訊更新 | 所有報名者 | 自訂更新內容 |
| 活動取消 | 所有報名者 | 自訂取消公告 |

## API 端點

### 活動
| 方法 | 路徑 | 說明 |
|---|---|---|
| GET | `/api/event` | 取得活動資訊 |
| PATCH | `/api/event` | 更新圖片 / 議程標題 |
| GET | `/api/agenda` | 取得議程列表 |

### 報名 / 會員
| 方法 | 路徑 | 說明 |
|---|---|---|
| POST | `/api/register` | 提交報名（自動建立會員） |
| POST | `/api/upload` | 上傳圖片（限 5MB） |
| POST | `/api/auth/login` | 登入，回傳 JWT |
| POST | `/api/auth/forgot-password` | 申請重設密碼 |
| POST | `/api/auth/set-password` | 設定新密碼（token） |
| GET | `/api/member/me` | 取得登入會員資料 |
| GET | `/api/member/registrations` | 取得報名紀錄 |
| POST | `/api/member/change-password` | 更改密碼 |

### 報到
| 方法 | 路徑 | 說明 |
|---|---|---|
| GET | `/api/checkin/qr` | 生成 QR Code PNG |
| GET | `/api/checkin/mark` | 掃碼標記出席（回傳 HTML） |

### 後台（需 Admin 密碼）
| 方法 | 路徑 | 說明 |
|---|---|---|
| GET | `/api/admin/members` | 取得所有會員（含報名資訊） |
| POST | `/api/admin/members` | 新增會員 |
| PATCH | `/api/admin/members/:id` | 更新會員 |
| DELETE | `/api/admin/members/:id` | 刪除會員 |
| DELETE | `/api/admin/members` | 清空所有會員 |
| POST | `/api/admin/notify-event-change` | 發送活動通知 |
| POST | `/api/admin/init-sheets` | 初始化 Google Sheets |

## 技術架構

| 項目 | 技術 |
|---|---|
| 前端 | React + Vite + React Router |
| 後端 | Node.js + Express |
| 資料儲存 | Google Sheets API（活動 / 報名 / 會員） |
| 認證 | JWT + bcryptjs |
| 信件 | Nodemailer + Gmail App Password |
| 圖片上傳 | Multer（存於 server/public/uploads） |
| 報到 QR | node-qrcode |
| Excel 匯出 | SheetJS (xlsx) |
| 前端部署 | Vercel |
| 後端部署 | Render（免費方案，閒置會休眠） |

## Google Sheets 結構

| 工作表名稱 | 說明 |
|---|---|
| `event-info-001` | 活動資訊（標題、日期、地點、說明、圖片 URL） |
| `agenda-001` | 議程（時間、主題、講師、說明、場次） |
| `registrations-001` | 報名紀錄（含報到狀態） |
| `members` | 會員資料（加密密碼、token 等） |
